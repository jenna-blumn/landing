## 비동기 메시지를 대화방에 연결하는 링크 기능 ##

### 기능 요약

- 비동기 채널 메시지는 그 메시지를 클릭한 사용자를 현재 상담중인 상담방에 연결하는 기능이 필요하다.
- 이것은 간단한 토글로 켜고 끌 수 있어야 한다.
- 토글을 켜면 메시지 발송시 상담방 연결 링크가 포함되어 발송된다.
- 토글을 켤때, 메시지 발송시 상담방 연결 링크가 포함되어 발송된다는 내용과 상담방이 닫혀 있으면 자동으로 새로운 상담방이 생성된다는 내용의 툴팁을 보여준다.

### UI 위치

- **컴포저 상단바(ComposerTopBar) 우측**, 기존 AI 토글(친절하게, 맞춤법) 옆에 배치
- pill 형태 버튼, 링크 아이콘 + "대화방 연결" 텍스트
- 활성: 파란색(soft/blue), 비활성: secondary 스타일

### 토글 노출 조건

- **비동기 채널(SMS, 알림톡, 이메일)에서만 표시**, 채팅 채널에서는 숨김
- `ChannelComposer`에서 `activeChannel !== 'chat'`일 때만 `roomLinkToggle`을 전달하고, 채팅일 때는 `undefined`를 전달하여 `ComposerTopBar`에서 렌더링하지 않음
- 채널을 전환하면 토글 버튼이 자동으로 나타나거나 사라짐

### 구현 상세

| 항목 | 내용 |
|------|------|
| 토글 상태 저장 | localStorage (`channelComposer_roomLink_v1`) |
| 기본값 | OFF (false) |
| 툴팁 | 토글 활성 시 hover하면 표시: "메시지 발송 시 대화방 연결 링크가 포함됩니다. 상담방이 닫혀있으면 새 상담방이 자동 생성됩니다." |
| 메시지 필드 | `ComposerMessage.roomLinkEnabled?: boolean` |
| API 옵션 | `SendMessageNowOptions.roomLinkEnabled?: boolean` |
