## 북마크 기능 ##

### 개요
고객 채팅 내용에 북마크 기능을 만든다. 전체 모드에서 동일하게 사용된다.

### 말풍선 북마크 아이콘

1. 각 말풍선에 마우스 오버하면 할일 생성 버튼 옆에 비활성 북마크 아이콘이 표시된다.
   - **고객 메시지**: `[말풍선] [할일 버튼] [북마크 버튼]` (우측)
   - **상담사 메시지**: `[북마크 버튼] [할일 버튼] [말풍선]` (좌측)
2. 작동 방법은 할일 생성 버튼과 유사:
   - 말풍선 마우스 오버 시 버튼 노출
   - 버튼 위 마우스 호버 시 색상 변화 (amber 계열)
3. 북마크를 체크하면 아이콘이 활성 상태로 계속 노출된다.
   - 활성 할일 버튼이 없으면 → 활성 북마크 버튼이 말풍선 바로 옆에 표시
   - 둘 다 활성이면 → 비활성 시 노출 순서대로 표시 (위 1번 참조)
   - 아이콘: DS `bookmark-3` (비활성: outline, 활성: fill, amber 색상)

### 북마크 리스트 버튼

- **위치**: 대화창(메시지 스크롤 영역) 우측 상단 플로팅 버튼
  - 반투명 배경 (`bg-white/80 backdrop-blur-sm`), hover 시 `bg-white/95`
  - 북마크가 있을 때만 표시되며, 북마크 개수 뱃지 포함
  - 행동과학 분석 결과: 대화창 내 네비게이션이므로 대화창 안에 배치하는 것이 맥락상 자연스러움
- **팝오버**: 버튼 아래쪽(`side="bottom"`, `align="end"`)으로 열림
  - 북마크된 말풍선의 내용을 최대 2줄로 표시 (`line-clamp-2`)
  - 발신자(고객/상담사) 및 시간 메타 정보 표시
  - 항목 클릭 시 해당 말풍선으로 스크롤 이동 (`scrollIntoView`)
  - 항목별 hover 시 북마크 해제(×) 버튼 표시

### 데이터 저장

- `bookmarkMap: Map<roomId, Set<messageId>>` 구조
- `localStorage` (`deskit-bookmarks` 키)에 영속화하여 새로고침 후에도 유지

### 적용 범위

- 전체 모드(채팅, 전화, 이력컨택, 스레드)에서 사용
